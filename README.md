DbAPI V0.1    (正在开发中)
===============


DbAPI为快速开发而生，其主要特性包括：

 + 提供WEB端的数据库管理功能
 + 提供HTTP接口对数据库进行增删查改
 + 其他功能逐渐增加中

> DbAPI的运行环境要求PHP5.4以上。

## 使用框架
前端：LayUi
后端：ThinkPhP
数据库支持：MySql
